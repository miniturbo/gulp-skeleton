# gulp skeleton

[gulp](http://gulpjs.com/) + [Browserify](http://browserify.org/) + [Bower](http://bower.io/) Skeleton

## Install

Install Node.js

```console
% brew install node
```

Install gulp

```console
% npm install -g gulp
```

Install npm packages

```console
% npm install
```

Install bower packages

```console
% bower install
```

## Usage

Build

```console
% gulp
```

Watch

```console
% gulp watch
```
